Postgrex.Decoder behaviour

Behaviour for custom decoding of postgres' binary format to elixir values.

Example

defmodule MyDecoder do
  use Postgrex.Decoder

  # Encoder of a direction enum
  def decode(:direction, _sender, _oid, value, nil) do
    case value do
      "up" -> :up
      "down" -> :down
    end
  end
end

Callbacks summary

decode(type :: atom(), sender :: atom(), oid :: integer(), value :: binary(), decoded :: term() | nil)

Decodes a binary from postgres' binary format to an elixir value. Also receives the decoded value as Postgrex decoded it or nil if it could not be decoded.
